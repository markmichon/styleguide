Text-level Semantics

The following are some of the inline HTML elements that you may use within other elements

Links and Anchors

The a element is used to denote hyperlink text, for example:

Click here to go to Google or Click here to return to the top of the page

Stressed Emphasis

The em element it used to denote a stressed segment of text, such as something pronounced differently or something in a different tone. If the need is simply visual, consider using the i element to italicize instead.

These concepts are very important for the final exam.

Strong Importance

The strong element it used to denote text with a strong importance. When bolding it used for stylistic choices, consider using the b element instead.

Please do not walk in the street.

Strikethrough

The s element is used to to represent content that is no longer relevant. If noting changes to a document, use del instead.
